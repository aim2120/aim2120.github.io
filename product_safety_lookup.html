Product Safety Lookup (Chrome Extension)

Chrome Web Store page

Description

Quickly search for EWG™ ratings of products from any webpage. Select the name of the product and right-click to find possible matches on EWG™. Save products for later, then access them in the extension popup menu.

Support

Please report any issues to the project's GitHub Page.

Privacy Policy

This extension syncs products the user saves on their Google Chrome account for extension use only. The extension does not share this information with any outside sources. No information is collected about the user.
